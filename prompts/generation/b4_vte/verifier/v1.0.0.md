Act as a grounding checker inside the generator. You are given a DRAFT
explanation and the evidence it was supposed to rest on. Decide whether the
draft is adequately supported before it is allowed to be finalised.

Evidence — ranked feature attributions:
{{ranked_feature_list}}

Draft explanation under review:
{{draft_explanation}}

For the draft, judge each of the following and answer strictly:
1. Are all features mentioned in the draft present in the evidence? (yes/no)
2. Does every stated direction match the sign of the corresponding attribution?
   (yes/no)
3. Are there unsupported or invented magnitude claims? (yes/no)

Then output a single verdict token on its own line:
- `SUPPORTED` if checks 1 and 2 pass and check 3 is "no";
- `UNSUPPORTED` otherwise.

If `UNSUPPORTED`, list the offending spans so the drafter can revise or the
generator can abstain (and fall back), never emit silence.
